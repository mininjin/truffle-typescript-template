# truffle-typescript-template

This repo is a typescript template for truffle.

## Commands

### Compile solidity files and create types of contracts

```
yarn typecheck
```

### Compile and copy into client

```
yarn compile
```

### Test

```
yarn test
```

### Migration

```
yarn migrate
```
